## Getting Started
   Expense Tracker Flutter App
   A comprehensive expense tracking application built with Flutter that allows users to manage their daily expenses with local storage persistence.


## Features

  Add Expenses** - Create new expense entries with title, amount, category, date, and description 
  View All Expenses** - See all recorded expenses in a organized list view
  Edit Expenses** - Modify existing expense details
  Delete Expenses** - Remove expenses with swipe-to-delete or edit screen option
  Local Storage** - All data persists using SharedPreferences
  Total Calculation** - Automatically calculates total expenses
  Form Validation** - Input validation for all fields
  Categories** - Pre-defined categories for better organization
  Responsive Design** - Works on different screen sizes
